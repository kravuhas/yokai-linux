PC Físico (Pendrive)
  • Baixe a ISO pelo MEGA.NZ
  • Baixe o Rufus (Windows) ou Balena Etcher (qualquer OS)
  • Plugue o pendrive (mín. 8 GB) e selecione a ISO
  • Reinicie o PC e pressione F12 / F2 / DEL no boot
  • Selecione o pendrive como dispositivo de boot
  • Sistema inicia no modo Live — sem instalar nada no HD
VM (VirtualBox / VMware)
  • Baixe a ISO pelo MEGA.NZ
  • Abra o VirtualBox ou VMware (ambos gratuitos)
  • Crie nova VM → Tipo: Linux → Versão: Arch Linux 64-bit
  • RAM: mínimo 4 GB recomendado · Disco: 20 GB
  • Em Armazenamento → monte a ISO como DVD
  • Inicie a VM — sistema carrega diretamente
